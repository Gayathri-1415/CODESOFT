# CodSoft Java Internship

This repository contains the tasks completed during my Java Programming Virtual Internship at CodSoft. The projects are developed using Java and focus on strengthening programming fundamentals, problem-solving skills, and object-oriented programming concepts.

## Tasks Completed

### Task 1: Number Game

A Java console-based game where the computer generates a random number, and the user tries to guess it within a limited number of attempts.

**Key Features:**

* Random number generation
* User input handling
* Guess validation
* Attempt tracking
* Interactive gameplay

### Task 2: Student Grade Calculator

A Java application that calculates total marks, average percentage, and grade based on the marks obtained in multiple subjects.

**Key Features:**

* Accepts marks for multiple subjects
* Calculates total marks
* Calculates average percentage
* Assigns grades automatically
* Displays complete result summary
